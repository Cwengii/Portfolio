# Acwengile Morris - Portfolio

A modern, creative portfolio showcasing my work as a Software & Data Engineer.

## 🚀 Quick Start

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## 🛠️ Built With

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## 📁 Project Structure

- `/src/components` - React components
- `/src/assets` - Images and static files
- `/src/pages` - Page components
- `/public` - Public assets (CV, etc.)

## 📝 Customization

### Update Projects
Edit `src/components/Projects.tsx` to add/modify projects

### Update About Section
Edit `src/components/About.tsx` for skills and bio

### Update Get to Know Me
Edit `src/components/GetToKnowMe.tsx` for interests and photos

### Add Images
Place images in `src/assets/` folder:
- `project-1.jpg`, `project-2.jpg`, `project-3.jpg` - Project images
- `interest-1.jpg`, `interest-2.jpg`, `interest-3.jpg` - Personal photos
- `hero-skate.jpg` - Hero background

### Add CV
Place your CV as `cv.pdf` in the `public/` folder

## 🌐 Deploy

Build for production:
```sh
npm run build
```
